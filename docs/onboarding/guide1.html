Before you begin

What you are about to do, and what you will need

Here is the whole journey in plain language. You will create a Claude account, turn on the paid plan that trajecktory needs, install the Claude app on your computer, and open the part of Claude called Code. That is it. The Code tab finishes coming to life in Guide 2: trajecktory installs a small helper called Git for you, and one restart switches it on.

The five steps in this guide
  1. Create your Claude account
  2. Turn on Claude Pro (the paid plan)
  3. Install the Claude app on your PC
  4. Open Claude Code (the Code tab)
  5. Point Claude at your work
What to have ready
  • A Windows PC (the one you will use for your job search)
  • An email address you can check
  • A credit or debit card for the Claude Pro plan
  • About 15 to 20 minutes of quiet time
⚠ One thing to know up front about cost. trajecktory's smartest features (reading job posts and scoring them) run on a paid Claude plan called Claude Pro. It costs $20 per month, or $17 per month if you pay for a year at once. The free version of Claude will not work for this. You can cancel any time. We will turn this on together in Step 2.

1

Create your Claude account

An account is just your personal sign in, like setting up a new email or shopping account.
  1. Open your web browser (Edge or Chrome), click the address bar at the very top, type claude.com and press Enter.
  2. On the page that appears, find the sign up card on the left. You can choose Continue with Google (use your Gmail) or Continue with email (use any email address).
  3. If you picked email, Claude sends you a message with a button or code to confirm it is really you. Open your email, click that button, and you are back in.

2

Turn on Claude Pro (the paid plan)

This unlocks the Claude features trajecktory relies on. It is the one paid part of all of this.
  1. While signed in, go to the address bar, type claude.com/pricing and press Enter.
  2. You will see three plans side by side: Free, Pro, and Max. Look at the middle one, Pro.
  3. Under Pro, click the dark Try Claude (or Get Claude) button and follow the screens to enter your card and confirm.
ⓘ Why Pro and not Free. trajecktory's one click "Evaluate" and "Scan" buttons, and the Code tool you will open in Step 4, all require a paid plan. Pro is the least expensive one that includes them. The bigger "Max" plan is only for very heavy daily use, so most people start with Pro.
💳
The payment screen
The checkout page is tied to your own account and card, so it is not shown here. Enter your details and confirm. Drop your own screenshot into this box if you want a record of it.
You can cancel any time. Pro is month to month. If you stop your search, you can turn it off from your account settings and you will not be charged again.
Keep the same account. Use the very same Claude account you made in Step 1 for the payment. Everything in both guides uses this one account.

3

Install the Claude app on your PC

Until now you have used Claude in a web browser. The app is a proper program on your computer, and it is where the Code tool lives.
  1. In the address bar, type claude.com/download and press Enter.
  2. Click the dark Download for Windows button. A file will save to your computer (usually to your Downloads folder).
  3. Open that downloaded file by double clicking it. The installer runs on its own. When it finishes, the Claude app opens.
  4. In the app, sign in with the same Claude account from Step 1.

Where did the file go? Downloads usually land in your Downloads folder. Open the folder icon on your taskbar, click Downloads on the left, and look for the file named like Claude at the top.

4

Open Claude Code (the Code tab)

This is the part people call "Claude Code mode". It is simply a tab inside the Claude app.

When the Claude app is open and you are signed in, look along the top center of the window. There are three tabs: Chat, Cowork, and Code. Click Code. That is the whole "switch to Claude Code" action. There is no separate mode to flip on. You can click it now to see it. It starts accepting prompts in Guide 2, once trajecktory installs Git and you restart your PC once.

ⓘ If clicking Code asks you to upgrade, it means the paid plan from Step 2 is not active yet. Finish turning on Claude Pro, then come back and click Code again.
↻ If it asks you to sign in, complete the sign in it shows, then fully close and reopen the Claude app to finish signing in. You will start working in the Code tab in Guide 2, once trajecktory is installed and you have restarted your PC.

5

Point Claude at your work

Telling Code which folder to work in. You will use this in Guide 2; here we just learn the moves.
  1. Inside the Code tab, choose Local. This simply means "use the files on this computer".
  2. Click Select folder and pick a folder. In Guide 2 you will pick your trajecktory folder; for now, just notice where this button is.
  3. From then on, you type what you want in plain English and Claude does it, showing you each change to approve first.
♥ You are always in control. By default, Claude Code asks before it changes anything. It shows you exactly what it wants to do with clear Accept and Reject buttons. Nothing on your computer changes until you click Accept. If you are ever unsure, you can just reject and ask Claude to explain.

Signing in to Claude (always the same account)

A lot of people worry about signing in over and over, so let us settle it now. You sign in to Claude here in the Claude app, which powers the Code tab. Later, when you first use trajecktory's one-click Evaluate and Scan buttons, it asks you to sign in to Claude one more time, with the same account. That is the only second sign-in, and you do it once.

🔗 The second sign-in is simple. When you first use trajecktory's Evaluate or Scan, you click Sign in to Claude in the sidebar and follow a few short prompts (Guide 2 walks it through, screen by screen). Same account, done once.

The Claude Code steps, on one card

If you ever need to get back into Claude Code from scratch, here is the whole path. These steps are confirmed against Claude's official help pages.

  1. Open the Claude app and sign in with your account.
  2. At the top center, click the Code tab (it sits next to Chat and Cowork).
  3. If it offers to upgrade, turn on Claude Pro first. If it asks you to sign in, do so and reopen the app.
  4. Choose Local, then click Select folder and choose your folder.
  5. On Windows, the Code tab opens your local files only after trajecktory is installed (it adds Git for you) and you restart your computer. That happens in Guide 2.
  6. Claude works in Ask permissions mode: it proposes each change and you click Accept or Reject.
♥ No computer commands, anywhere. Across this entire guide you never had to type a single technical command. Everything was clicking buttons and links. That stays true in Guide 2 as well.
